# Topic taxonomy

Closed list of tags used to classify per-paper notes. Revisable: add/rename tags here before applying them. Each note declares 1–4 of these.

## Optimization framework

- `[mordm]` — Many-Objective Robust Decision Making (the framework as a whole).
- `[borg-moea]` — Borg algorithm internals, operators, adaptation.
- `[mm-borg]` — Multi-Master parallelization variant.
- `[moea-diagnostics]` — convergence, hypervolume, attainment, seed reliability.
- `[moea-comparison]` — head-to-head MOEA performance studies.

## Policy formulation

- `[direct-policy-search]` — EMODPS and related.
- `[parameterized-rules]` — operating-rule parameterization (FFMP, zone-based).
- `[rbf-policy]` — radial basis function policies.
- `[ann-policy]` — neural-network policies.
- `[tree-policy]` — decision-tree policies (hard or soft).
- `[spline-policy]` — spline / additive policies.
- `[policy-formulation-comparison]` — papers explicitly comparing policy classes.

## Robustness & uncertainty

- `[robustness-metrics]` — satisficing, regret, domain criterion, signal-to-noise.
- `[deep-uncertainty]` — DMUU framing, scenario taxonomies.
- `[multi-stakeholder-equity]` — equity in robustness across decision-makers.

## Scenario analysis

- `[scenario-discovery]` — finding policy-relevant uncertainty regions.
- `[prim-cart]` — classical scenario-discovery methods.
- `[explainable-ai-shap-lime]` — SHAP/LIME applied to water-resources policy analysis.
- `[narrative-storylines]` — qualitative scenario storylines (FRNSIC and similar).

## Stochastic generation

- `[stochastic-streamflow]` — synthetic streamflow ensemble generation.
- `[kirsch-nowak]` — the Kirsch & Nowak generator family specifically.
- `[climate-perturbed]` — climate-conditional scenario generation.

## DRB / system context

- `[drb-ffmp]` — Delaware River Basin operations, FFMP, decree parties.
- `[reservoir-operations]` — reservoir-operations policy and modeling more broadly.
- `[salt-front]` — salt-front intrusion in the Delaware estuary.
- `[water-temperature]` — Lordville thermal stress, cold-water habitat.
- `[pywrdrb]` — Pywr-DRB simulation model.

## Methods (cross-cutting)

- `[ml-in-water-systems]` — machine learning applied to water resources broadly.
- `[lstm-hydro]` — LSTM applications to streamflow, salinity, temperature.
- `[surrogate-modeling]` — Gaussian processes, random forests as policy/sim surrogates.

## Background / canon

- `[canonical]` — foundational citation that everyone in the area knows. Tag sparingly — for true canon only.
